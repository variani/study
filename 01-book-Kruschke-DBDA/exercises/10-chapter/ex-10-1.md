## Exercise 10.1

Functions:


```r
pD <- function(z, N, a, b) exp(lbeta(z + a, N - z + b) - lbeta(a, b))
```

### Part A


```r
z <- 7
N <- 10

w1 <- 0.25
w2 <- 0.75
k <- 6

a1 <- w1 * (k - 2) + 1
b1 <- (1 - w1) * (k - 2) + 1

a2 <- w2 * (k - 2) + 1
b2 <- (1 - w2) * (k - 2) + 1

(pD1 <- pD(z, N, a1, b1))
```

```
[1] 0.0004440004
```

```r
(pD2 <- pD(z, N, a2, b2))
```

```
[1] 0.001332001
```

```r
(BF <- pD2 / pD1)
```

```
[1] 3
```
